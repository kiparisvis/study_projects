3DViewer_v1.0

О программе

3DViewer_v1.0 позволяет визуализировать 3D модель. 3D-визуализацией называют процесс, в ходе которого осуществляется обработка и рендеринг (отрисовка) элементов компьютерной графики (объектов, сцен) с применением специального ПО и инструментария, предназначенного для работы c 3D.

Установка

Для того, чтобы начать использовать 3DViewer_v1.0, вам нужно использовать команду make install.

Удаление

Для удаления программы вы можете использовать команду make uninstall.

Как начать работу с программой?

    • 1. Перейти в каталог 3d_viewer/src;
    • 2. Ввести в консоль команду make;
    • 3. После установки программы откроется окно. Выбрать любую модель из папки obj;
    • 4. Откроется программа с отрисованной моделью. Если модели не видно, уменьшите масштаб колесом мыши.

Программа позволяет:

    • Настраивать тип проекции (параллельная и центральная);
    • Настраивать тип ребер (сплошная, пунктирная);
    • Настраивать цвет и толщину ребер;
    • Настраивать способ отображения вершин (отсутствует, круг, квадрат);
    • Настраивать цвет и размер вершин;
    • Перемещать модель на заданное расстояние относительно осей X, Y, Z;
    • Поворачивать модель на заданный угол относительно своих осей X, Y, Z;
    • Масштабировать модель на заданное значение;
    • Сохранять полученные изображения в файл в форматах bmp и jpeg;
    • Записывать небольшие "скринкасты" загруженного объекта в gif-анимацию;
    • Работать с файлами типа *.obj.

Технические требования для запуска программы:

    • Установленный фреймворк OpenGL;
    • Установленный и настроенный фреймворк QtCreator версии 6.1;
    • Операционная система Mac Os.
