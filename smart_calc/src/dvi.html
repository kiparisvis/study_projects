Smart_calc

О программе:

Smart_calc позволяет производить стандартные математические расчеты с некоторыми добавлениями в функционале.

Установка

Для того, чтобы начать использовать Smart_calc, вам нужно использовать команду make install.

Удаление

Для удаления программы вы можете использовать команду make uninstall.

Как начать работу с программой?

    • Перейти в каталог smart_calc/src;
    • Ввести в консоль команду make;
    • После установки программы откроется окно калькулятора;
    • Для построения графика необходимо ввести функцию и задать максимальные значения по оси X и Y. Например: ввести в поле для ввода функцию sin(x), ниже установить Xmix = -10; Xmax = 10; Ymin = -1; Ymax = 1 и нажать кнопку "Graph";
    • Для использования функционала кредитного калькулятора необходимо открыть вкладку "Кредитный калькулятор". В открывшемся окне внести необходимые параметры для расчета и нажать кнопку "Рассчитать";
    • Для использования функционала депозитного калькулятора необходимо открыть вкладку "Депозитный калькулятор". В открывшемся окне внести необходимые параметры для расчета и нажать кнопку "Рассчитать"

Программа позволяет:

    • Производить стандартные математические операции:
      • - Сложение: +;
      • - Вычитание: -;
      • - Умножение: *;
      • - Деление: /;
      • - Возведение в степень: ^;
      • - Нахождение квадратного корня: sqrt(9);
      • - Нахождение остатка от деления на число: 6mod3;
      • - Вычисление натурального логарифма: ln(6);
      • - Вычисление десятичного логарифма: log(10);
      • - Реализована приоритетность математических операций.
    • Построение графиков функций с возможностью указать максимальные и минимальные значение координат для графика:
      • sin(x);
      • cos(x);
      • tan(x)
      • asin(x);
      • acos(x);
      • atan(x)
    • Производить расчет ежемесячных платежей по кредиту в режиме "кредитный калькулятор";
    • Производить расчет ежемесячных процентов по вкладу в режиме "депозитный калькулятор";

Технические требования для запуска программы:

    • Установленный и настроенный фреймворк QtCreator версии 6.1;
    • Операционная система Mac Os.
